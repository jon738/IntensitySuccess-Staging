The Momentum Framework

Read time: ~5 min · Posted Oct 21, 2025

Momentum is a design choice. If you can reliably create a clean start, a clear next step, and a quick win, you’ll move — even on low-motivation days. The Momentum Framework compresses that into three loops you can run daily.

Loop 1 — Start Clean

  • Clear the deck (2 min): jot every open loop on a scratch pad.
  • Pick the lever: choose one task that makes others easier or irrelevant.
  • Prime the first keystroke: write one sentence of what you’ll do next.

Loop 2 — Move Small

Work in tiny, finishable units so you get the dopamine of done:

  • Scope to 25 minutes or less.
  • Define “done” in one line: “Send draft to X,” “Merge PR,” “Publish post.”
  • Capture the next breadcrumb before you stop.

Loop 3 — Close + Log

  • Move the card to Done (make progress visible).
  • Write a 2-line log: What moved? and What’s first tomorrow?
  • Reset your workspace so the next start is frictionless.

Copy-Paste Card Template

Task: ____________________________
Definition of Done: _______________
First Keystroke: _________________
Block Size: 25m
Next Breadcrumb (when stopping): __

Daily Score (simple)

[ ] 2 blocks shipped
[ ] 1 thing sent (draft/PR/email)
[ ] Close loops + set first step

Run the three loops. Momentum compounds because starts get easier, wins come faster, and your system — not your mood — drives the day.
